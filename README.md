# Instapicapp

Aplicación móvil inspirada en "Instagram" desarrollada con React Native, diseñada para replicar una experiencia moderna de red social móvil mediante una interfaz intuitiva y componentes reutilizables.
El proyecto se centra en el diseño responsive, la navegación fluida y la organización escalable de componentes para aplicaciones móviles.

Funcionalidades
- Feed de publicaciones
- Interfaz inspirada en redes sociales 
- Navegación entre pantallas
- Componentes reutilizables
- Diseño responsive
- Gestión de estados
  
Tecnologías utilizadas
- React Native
- JavaScript
- React Navigation
- Styled Components / CSS
  
Objetivos del proyecto: practicar la arquitectura de las app móviles, mejorar el diseño UI / UX, implementar componentes reutilizables y similar las funcionalidades de una red social


## Como ejecutar e iniciar la aplicación




```http
  npm install
```

```http
  npm start
```

Una vez iniciada la aplicación ingresar el enlace de localhost que sale en la linea de comandos
